## VS Code

<Image src="/docs/integration/vscode.gif" alt="VS Code extension DEMO" />

Although scaffdog can be used as a CLI tool, this extension allows you to use scaffdog intuitively from the GUI :dog:

- [scaffdog - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=scaffdog.scaffdog-vscode)

For more information on this extension, please refer to [here](https://github.com/scaffdog/vscode).

## Prettier

We provide a Prettier plugin that automatically formats scaffdog templates :dog: :nail_care:

**Install via npm:**

```bash
$ npm install --save-dev prettier-plugin-scaffdog
```

For more information on this plugin, please refer to [here](https://github.com/scaffdog/scaffdog/tree/canary/packages/prettier-plugin-scaffdog#readme).
